🛠 Умови виконання  
•	Використовувати лише HTML + CSS

### Завдання  
1️⃣  Контейнер - min()  
Реалізуйте контейнер так, щоб:  
•	його максимальна ширина була 1200px  
•	з обох боків були внутрішні відступи по 16px  
•	контейнер був вирівняний по центру сторінки  
•	ширина була адаптивною  
Використати функцію min(). 
________________________________________  
2️⃣ Заголовок h1 має:  
* мінімальний розмір — 24px  
•	масштабуватися залежно від ширини екрана  
•	максимальний розмір — 48px  
•	бути вирівняним по центру  
•	мати верхній відступ 50px та нижній 30px  
📌 Використати функцію clamp().  
________________________________________  
3️⃣   Блок карток (Flexbox)  
Секція .cards повинна:  
•	використовувати display: flex  
•	вирівнювати картки по центру  
•	мати відступ між картками 20px  
•	дозволяти перенос елементів на новий рядок при зменшенні ширини екрана  
________________________________________  
4️⃣   Картки - clamp()  
Кожна картка повинна:  
•	мати мінімальну ширину — 150px  
•	мати максимальну ширину — 300px  
•	гнучко змінювати розмір між цими значеннями  
•	мати внутрішній відступ — 16px  
•	рамку товщиною 3px  
•	заокруглення кутів — 12px  
📌 Використати clamp() для властивості width.  
________________________________________  
5️⃣   Розмір тексту в картках (clamp())  
Текст у .card повинен:  
•	мати мінімальний розмір — 14px  
•	плавно збільшуватися на більших екранах  
•	не перевищувати 20px  
📌 Використати clamp() для font-size.  
________________________________________  
### Очікувана адаптивна поведінка   
•	На великих екранах картки розташовані в один ряд.    
•	При зменшенні ширини вони переходять у 2 в ряд.  
•	На вузьких екранах — по 1 в ряд.  
•	Жодних @media не використовується.  

## CSS   
.container {  
  width: min();  
}  
h1 {  
  font-size: clamp();  
}

}

  font-size: clamp();

  }
